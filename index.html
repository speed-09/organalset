async function testGeminiAPI() { const apiKey = "သင့်ရဲ့_API_KEY_ကိုဒီမှာထည့်ပါ"; const url = `https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent?key=${apiKey}`; const requestBody = { contents: [{ parts: [{ text: "နေကောင်းလား" }] }] }; try { const response = await fetch(url, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(requestBody) }); const data = await response.json(); if (response.ok) { console.log("အောင်မြင်ပါသည် - ", data.candidates[0].content.parts[0].text); alert("API အလုပ်လုပ်နေပါပြီ!"); } else { console.error("Error Detail:", data); alert("Error: " + data.error.message); } } catch (error) { console.error("Fetch Error:", error); alert("ချိတ်ဆက်မှု မအောင်မြင်ပါ (VPN စစ်ဆေးပါ)"); } }
